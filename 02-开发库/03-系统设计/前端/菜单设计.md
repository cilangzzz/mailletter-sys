## 七、菜单交互规则

### 7.1 菜单展开/折叠

| 触发条件 | 响应行为 |
|----------|----------|
| 点击顶部导航菜单 | 切换到对应模块，侧边菜单显示该模块下的子菜单 |
| 点击侧边菜单（有子菜单） | 展开/折叠子菜单列表 |
| 点击侧边菜单（无子菜单） | 直接跳转对应页面 |
| 点击子菜单项 | 跳转对应页面，保持父菜单展开状态 |
| 进入页面 | 自动展开并高亮对应菜单项 |

### 7.2 菜单高亮规则

| 场景 | 高亮规则 |
|------|----------|
| 当前页面 | 对应菜单项高亮显示 |
| 子菜单页面 | 子菜单项高亮，父菜单项次高亮 |
| 面包屑联动 | 面包屑路径与菜单层级一致 |

### 7.3 权限控制

| 权限类型 | 控制方式 |
|----------|----------|
| 菜单可见性 | 根据用户角色权限显示/隐藏菜单项 |
| 菜单可访问性 | 无权限菜单点击提示"无访问权限" |
| 动态菜单 | 后端返回用户可访问菜单列表，前端动态渲染 |

### 7.4 响应式适配

| 设备类型 | 菜单表现 |
|----------|----------|
| 桌面端 | 顶部导航固定，侧边栏固定展开 |
| 平板端 | 侧边栏可折叠，图标+文字或纯图标 |
| 手机端 | 侧边栏隐藏，点击汉堡菜单展开抽屉 |

## 九、菜单图标规范

### 图标来源
- 使用 Ant Design Icons 或 Element Plus Icons
- 保持图标风格统一（线性图标或填充图标）

### 图标尺寸
- 顶部导航菜单：20px × 20px
- 侧边一级菜单：18px × 18px
- 侧边二级菜单：可选，默认不显示

### 图标颜色
- 默认状态：菜单文字颜色
- 高亮状态：主题色
- 禁用状态：灰色
