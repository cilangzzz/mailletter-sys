# 留言统计 - UI设计

> 本文档记录留言统计页面的UI设计信息，周统计和月统计通过Tab切换。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 留言统计 |
| 页面标识 | statistics |
| 所属模块 | M01-信件核心模块 |
| 页面类型 | 统计页 |
| 路由路径 | /statistics |

## 三、Tab切换设计

| Tab名称 | 默认选中 | 说明 |
|--------|----------|------|
| 周统计 | 是 | 每周数据统计 |
| 月统计 | 否 | 每月数据统计 |

**交互说明：**
- 点击Tab切换显示周统计或月统计内容
- 切换后数据自动更新

---

## 四、周统计内容

### 4.1 时间选择

| 元素名称 | 说明 |
|----------|------|
| 当前周显示 | 显示当前周的范围（如：2026年4月第2周） |
| 上一周按钮 | 切换到上一周数据 |
| 下一周按钮 | 切换到下一周数据 |

### 4.2 统计卡片

| 卡片名称 | 数据说明 |
|----------|----------|
| 本周来信 | 本周新增信件总数 |
| 本周受理 | 本周受理信件数 |
| 本周完成 | 本周完成信件数 |
| 本周超期 | 本周超期信件数 |
| 本周退回 | 本周退回信件数 |

### 4.3 图表区

| 图表名称 | 图表类型 | 说明 |
|----------|----------|------|
| 来信趋势折线图 | 折线图 | 周一到周日来信数量变化 |
| 来信类型分布饼图 | 饼图 | 按来信类型分布统计 |
| 各单位办理情况柱状图 | 柱状图 | 各单位办理数量对比 |

### 4.4 数据表格

| 表头字段 | 说明 |
|----------|------|
| 单位名称 | 办理单位名称 |
| 本周受理 | 本周受理信件数 |
| 本周完成 | 本周完成信件数 |
| 完成率 | 完成率百分比 |
| 平均回复时长 | 平均回复时间（小时） |
| 是否超期 | 超期标记 |

---

## 六、导出功能

| 按钮名称 | 说明 |
|----------|------|
| 导出周报 | 导出周度统计报告 |
| 导出月报 | 导出月度统计报告 |
| 导出详细数据 | 导出详细统计数据Excel |

---

## 七、交互规则

| 触发条件 | 响应行为 |
|----------|----------|
| 点击Tab切换 | 切换显示周统计/月统计内容 |
| 点击上一周/下一周 | 切换显示对应周的数据 |
| 点击上一月/下一月 | 切换显示对应月的数据 |
| 点击图表区域 | 可查看详细数据或放大查看 |
| 点击导出按钮 | 导出对应格式数据 |
