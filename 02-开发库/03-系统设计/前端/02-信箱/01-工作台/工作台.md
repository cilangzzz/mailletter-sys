## 四、交互规则

| 触发条件 | 响应行为 |
|----------|----------|
| 点击待办数量 | 跳转到对应状态的信件列表 |
| 点击图表区域 | 可查看详细数据或放大 |
| 点击查看详情 | 跳转到信件详情页 |
| 点击处理按钮 | 跳转到信件详情页进行处理 |
| 页面加载 | 自动刷新统计数据 |

---

## 五、权限说明

| 用户角色 | 显示内容 |
|----------|----------|
| 普通办理员 | 本单位相关统计数据 |
| 管理员 | 全局统计数据 |
| 系统管理员 | 全局统计数据 + 后台管理入口 |

## 六、样式说明

### 6.1 统计卡片样式

| 元素 | 样式 |
|------|------|
| 卡片背景 | 浅色背景，圆角边框 |
| 数字字体 | 24px bold，主题色 |
| 标签字体 | 14px normal，灰色 |
| 大卡片 | 突出显示，较大字号 |

### 6.2 待办项样式

| 状态 | 样式 |
|------|------|
| 正常 | 蓝色数字 |
| 超期 | 红色数字，高亮显示 |
| hover | 下划线提示可点击 |

### 6.3 响应式适配

| 断点 | 布局调整 |
|------|----------|
| ≥1200px | 四列统计卡片，图表并排显示 |
| 768-1199px | 两列统计卡片，图表堆叠 |
| <768px | 单列显示，图表隐藏或简化 |
