# 上级指派信件 - UI设计

> 本文档记录上级指派信件列表页的UI设计信息，信件类型通过按钮筛选。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 上级指派信件 |
| 页面标识 | letter-assigned |
| 所属模块 | M01-信件核心模块 |
| 页面类型 | 列表页 |
| 路由路径 | /mailbox/letter/assigned |

## 三、页面元素清单

### 3.1 信件类型筛选按钮

| 按钮名称 | 说明 |
|----------|------|
| 厅长信箱指派 | 上级通过厅长信箱指派下来的信件 |
| 信访件指派 | 上级通过信访系统指派下来的信件 |
| 12389指派 | 上级通过12389平台指派的投诉件 |
| 全部指派 | 所有上级指派信件汇总 |

### 3.2 搜索条件

| 字段名称 | 字段标识 | 输入类型 | 是否必填 | 说明 |
|----------|----------|----------|----------|------|
| 留言ID | letter_id | 数字输入 | 否 | 精确匹配 |
| 受理编号 | process_no | 文本输入 | 否 | 模糊匹配 |
| 留言标题 | letter_title | 文本输入 | 否 | 模糊匹配 |
| 指派单位 | assign_unit | 下拉选择 | 否 | 指派来源单位 |
| 主办单位 | main_unit | 下拉选择 | 否 | 本单位作为主办 |
| 办理状态 | process_status | 下拉选择 | 否 | 办理状态选项 |
| 指派时间 | assign_time | 日期范围选择 | 否 | 指派时间范围 |
| 答复时间 | reply_time | 日期范围选择 | 否 | 答复时间范围 |

### 3.3 操作按钮

| 按钮名称 | 按钮类型 | 说明 |
|----------|----------|------|
| 批量操作 | 下拉按钮 | 批量导出/批量答复 |
| 刷新 | 图标按钮 | 刷新当前列表 |
| 导出 | 下拉按钮 | 导出指派信件数据 |

### 3.4 状态标签页

| 标签页名称 | 数量示例 | 说明 |
|----------|----------|------|
| 全部 | 200 | 所有上级指派信件 |
| 待回复 | 50 | 等待回复 |
| 已回复 | 100 | 已提交答复 |
| 已退回 | 20 | 已退回办理表/答复 |
| 超期未回复 | 30 | 超期未答复 |

### 3.5 数据列表

| 表头字段 | 是否可排序 | 说明 |
|----------|------------|------|
| checkbox | - | 全选/单选 |
| 单位角色 | 否 | 主办/督办/协办标签 |
| 留言ID | 否 | 关联留言编号 |
| 留言标题 | 否 | 留言标题 |
| 指派时间 | 是 | 指派办理时间 |
| 要求回复时间 | 是 | 时限要求 |
| 实际回复时间 | 是 | 答复时间 |
| 办理状态 | 否 | 未提交/已提交/已退回 |
| 答复状态 | 否 | 未提交/已提交/已退回 |
| 是否超期 | 否 | 超期标签（红色高亮） |
| 指派单位 | 否 | 上级指派单位名称 |
| 主办单位 | 否 | 本单位名称 |
| 操作 | - | 操作按钮 |

**每行操作按钮：**
| 按钮名称 | 按钮类型 | 说明 |
|----------|----------|------|
| 查看详情 | 文字按钮 | 跳转到信件详情页 |
| 查看办理信息 | 文字按钮 | 打开办理信息弹窗 |
| 提交答复 | 文字按钮(蓝色) | 打开答复弹窗 |

## 四、与本站点信件的区别

| 对比项 | 本站点信件 | 上级指派信件 |
|--------|------------|--------------|
| 数据来源 | 本站点直接接收 | 上级单位指派下来 |
| 操作权限 | 可指派、可删除 | 仅可答复、查看 |
| 状态分类 | 12种状态 | 5种状态 |
| 搜索字段 | 来信人信息为主 | 指派信息为主 |

---

## 五、交互规则

### 5.1 信件类型筛选

| 触发条件 | 响应行为 |
|----------|----------|
| 点击类型筛选按钮 | 切换显示对应来源的指派信件 |

### 5.2 提交答复

| 触发条件 | 响应行为 |
|----------|----------|
| 点击提交答复 | 打开答复弹窗，填写答复内容 |

### 5.3 查看办理信息

| 触发条件 | 响应行为 |
|----------|----------|
| 点击查看办理信息 | 打开办理信息弹窗，显示完整办理记录 |
