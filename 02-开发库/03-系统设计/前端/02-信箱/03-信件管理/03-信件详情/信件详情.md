## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 信件详情 |

---

## 二、页面布局（按状态区分）

信件详情页根据信件状态展示不同的布局和操作区域，主要分为三种状态类型：

### 2.1 整体布局框架

采用**上下分区布局**，顶部为留言信息（固定），下方根据状态动态展示不同区块。

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  状态标签区：[待受理] [受理中] [待审核] [已完成] [无法受理] ...              │
├─────────────────────────────────────────────────────────────────────────────┤
│  留言信息区块（固定展示）                                                     │
│  ┌─────────────────────────────────────────────────────────────────────────┐│
│  │ 状态标签 | 留言标题 | 留言ID | 留言类型                                   ││
│  │ 姓名 | 身份证号(可隐藏) | 电话(可隐藏) | 留言时间                         ││
│  │ 事发市(区/县) | 联系地址 | 关联信息                                       ││
│  │ 该用户留言总计xx条 | 重复留言总计xx条                                     ││
│  │ 留言详情内容 | 附件列表                                                   ││
│  │ [查看流转记录] [打印留言]                                                 ││
│  └─────────────────────────────────────────────────────────────────────────┘│
├─────────────────────────────────────────────────────────────────────────────┤
│  动态区块（根据状态变化）                                                     │
│  ┌─────────────────────────────────────────────────────────────────────────┐│
│  │ 留言属性区块 / 流转答复区块 / 指派处理区块 / 批示意见区块 / 本站答复区块 ││
│  └─────────────────────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────────────────────┘
```

## 八、交互规则

### 8.1 状态切换

| 触发条件 | 响应行为 |
|----------|----------|
| 点击状态标签页 | 切换列表显示对应状态信件 |
| 信件状态变更 | 页面自动刷新或提示刷新 |

### 8.2 详情页跳转

| 触发条件 | 响应行为 |
|----------|----------|
| 点击信件行 | 跳转到信件详情页 |
| 点击操作按钮 | 执行对应操作或弹窗 |

### 8.3 答复操作

| 触发条件 | 响应行为 |
|----------|----------|
| 点击编辑答复 | 打开答复编辑弹窗 |
| 点击退回办理表 | 确认后退回办理表 |
| 点击退回答复 | 确认后退回答复内容 |

### 8.4 批示意见操作

| 触发条件 | 响应行为 |
|----------|----------|
| 点击编辑批示意见 | 打开批示编辑弹窗 |
| 点击附件下载 | 下载批示意见附件 |
