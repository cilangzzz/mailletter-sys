## 三、新增/编辑弹窗

| 字段名称 | 输入类型 | 是否必填 |
|----------|----------|----------|
| 用户名 | 文本输入 | 是 |
| 姓名 | 文本输入 | 是 |
| 所属单位 | 下拉选择 | 是 |
| 角色 | 下拉选择 | 是 |
| 手机号 | 文本输入 | 是 |
| 邮箱 | 文本输入 | 否 |
| 状态 | 单选 | 是 |
