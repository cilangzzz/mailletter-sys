# 回复模板管理 - UI设计

> 本文档记录回复模板管理页面的UI设计信息。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 回复模板管理 |
| 页面标识 | reply-template |
| 所属模块 | M03-后台管理模块 |
| 页面类型 | 配置页 |
| 路由路径 | /backend/reply-template |

## 三、新增/编辑弹窗

| 字段名称 | 输入类型 | 是否必填 |
|----------|----------|----------|
| 模板名称 | 文本输入 | 是 |
| 模板类型 | 下拉选择 | 是 |
| 适用单位 | 下拉选择 | 否 |
| 模板内容 | 富文本编辑 | 是 |
| 状态 | 单选 | 是 |

---

## 四、模板类型

| 类型名称 | 说明 |
|----------|------|
| 公共模板 | 所有单位可用 |
| 业务模板 | 特定业务类型可用 |
| 单位模板 | 特定单位可用 |
