# 案事件指向管理 - UI设计

> 本文档记录案事件指向管理页面的UI设计信息。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 案事件指向管理 |
| 页面标识 | case-direction |
| 所属模块 | M03-后台管理模块 |
