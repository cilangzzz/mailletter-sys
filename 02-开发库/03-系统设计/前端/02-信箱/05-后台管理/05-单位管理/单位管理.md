# 单位管理 - UI设计

> 本文档记录单位管理页面的UI设计信息。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 单位管理 |
| 页面标识 | unit |
| 所属模块 | M03-后台管理模块 |
| 页面类型 | 配置页 |
| 路由路径 | /backend/unit |

## 三、新增/编辑弹窗

| 字段名称 | 输入类型 | 是否必填 |
|----------|----------|----------|
| 单位名称 | 文本输入 | 是 |
| 单位编码 | 文本输入 | 是 |
| 上级单位 | 下拉选择 | 否 |
| 单位类型 | 下拉选择 | 是 |
| 联系电话 | 文本输入 | 否 |
| 联系地址 | 文本输入 | 否 |
| 状态 | 单选 | 是 |
