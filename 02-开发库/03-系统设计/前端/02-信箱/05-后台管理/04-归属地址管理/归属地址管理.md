# 归属地址管理 - UI设计

> 本文档记录归属地址管理页面的UI设计信息。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 归属地址管理 |
| 页面标识 | address |
| 所属模块 | M03-后台管理模块 |
| 页面类型 | 配置页 |
| 路由路径 | /backend/address |

## 三、新增/编辑弹窗

| 字段名称 | 输入类型 | 是否必填 |
|----------|----------|----------|
| 地区名称 | 文本输入 | 是 |
| 地区编码 | 文本输入 | 是 |
| 上级地区 | 下拉选择 | 否（顶级不选） |
| 地区层级 | 单选 | 是（省/市/区县） |
| 状态 | 单选 | 是 |
