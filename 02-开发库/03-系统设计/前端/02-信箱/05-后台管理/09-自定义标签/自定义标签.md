# 自定义标签 - UI设计

> 本文档记录自定义标签管理页面的UI设计信息。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 自定义标签 |
| 页面标识 | custom-tag |
| 所属模块 | M03-后台管理模块 |
| 页面类型 | 配置页 |
| 路由路径 | /backend/custom-tag |
