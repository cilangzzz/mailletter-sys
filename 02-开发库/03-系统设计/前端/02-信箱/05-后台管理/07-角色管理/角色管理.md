# 角色管理 - UI设计

> 本文档记录角色管理页面的UI设计信息。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 角色管理 |
| 页面标识 | role |
| 所属模块 | M03-后台管理模块 |
| 页面类型 | 配置页 |
| 路由路径 | /backend/role |

## 三、新增/编辑弹窗

| 字段名称 | 输入类型 | 是否必填 |
|----------|----------|----------|
| 角色名称 | 文本输入 | 是 |
| 角色编码 | 文本输入 | 是 |
| 角色类型 | 下拉选择 | 是 |
| 状态 | 单选 | 是 |
| 备注 | 文本输入 | 否 |
