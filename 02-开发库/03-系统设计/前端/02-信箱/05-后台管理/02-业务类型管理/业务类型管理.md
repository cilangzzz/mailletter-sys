# 业务类型管理 - UI设计

> 本文档记录业务类型管理页面的UI设计信息。

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 业务类型管理 |
| 页面标识 | business-type |
| 所属模块 | M03-后台管理模块 |
| 页面类型 | 配置页 |
| 路由路径 | /backend/business-type |

---

## 二、页面布局

与来信类型管理页面类似，支持树形层级结构。

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  面包屑：首页 / 信箱 / 后台管理 / 业务类型管理                                │
├─────────────────────────────────────────────────────────────────────────────┤
│  操作按钮区：[新增类型] [新增子类型] [刷新]                                   │
├─────────────────────────────────────────────────────────────────────────────┤
│  树形列表区                                                                  │
│  ┌─────────────────────────────────────────────────────────────────────────┐│
│  │ ▼ 交通管理                                                               ││
│  │    ├─ 交通违章                                                           ││
│  │    ├─ 交通设施                                                           ││
│  │    └─ 交通拥堵                                                           ││
│  │ ▼ 户籍管理                                                               ││
│  │    ├─ 户口迁移                                                           ││
│  │    ├─ 身份证办理                                                         ││
│  │    └─ 居住证办理                                                         ││
│  │ ... 更多类型 ...                                                         ││
│  └─────────────────────────────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────────────────────────────┘
```

---

## 三、页面元素清单

### 3.1 操作按钮

| 按钮名称 | 说明 |
|----------|------|
| 新增类型 | 新增顶级业务类型 |
| 新增子类型 | 新增子级业务类型 |
| 刷新 | 刷新列表 |

### 3.2 树形列表

| 字段名称 | 说明 |
|----------|------|
| 类型名称 | 业务类型名称 |
| 层级 | 树形层级展示 |
| 操作 | 编辑/删除/新增子类型 |

---

## 四、新增/编辑弹窗

| 字段名称 | 输入类型 | 是否必填 |
|----------|----------|----------|
| 类型名称 | 文本输入 | 是 |
| 上级类型 | 下拉选择 | 否（顶级类型不选） |
| 排序 | 数字输入 | 否 |
| 状态 | 单选 | 是 |
