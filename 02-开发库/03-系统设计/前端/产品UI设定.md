# 产品UI设定规范

## 一、页面基本信息

| 属性 | 说明 | 示例 |
|------|------|------|
| 页面名称 | 用户可见的页面标题 | 信件列表 |
| 页面标识 | 唯一标识符，用于路由和代码引用 | letter-list |
| 所属模块 | 功能模块归属 | M01-信件核心模块 |
| 页面类型 | 列表页/详情页/表单页/仪表盘/配置页 | 列表页 |

### 页面类型定义

| 类型 | 说明 | 典型特征 |
|------|------|----------|
| 列表页 | 数据集合展示与筛选 | 表格/卡片列表、搜索、批量操作 |
| 详情页 | 单条数据完整信息展示 | 字段分组、关联数据、操作按钮 |
| 表单页 | 数据录入与编辑 | 表单验证、提交保存、取消重置 |
| 仪表盘 | 数据统计与可视化 | 图表、指标卡片、趋势分析 |
| 配置页 | 系统参数设置 | 配置项列表、开关、保存按钮 |

## 二、页面布局结构

### 标准布局框架

```
┌─────────────────────────────────────────────────────────┐
│  顶部导航栏 (Header)                                      │
│  - Logo、系统名称、全局搜索、用户信息、通知入口               │
├───────────┬─────────────────────────────────────────────┤
│           │  页面标题区 (Page Header)                     │
│  侧边栏    │  - 页面标题、面包屑导航、主要操作按钮             │
│  (Sidebar)├─────────────────────────────────────────────┤
│           │                                             │
│  - 导航菜单│  内容区域 (Content Area)                      │
│  - 模块入口│  - 筛选/搜索区                               │
│           │  - 数据展示区                                │
│           │  - 分页/加载区                               │
│           │                                             │
├───────────┴─────────────────────────────────────────────┤
│  底部栏 (Footer) - 版权信息、系统版本                        │
└─────────────────────────────────────────────────────────┘
```

### 区域说明

| 区域 | 高度比例 | 包含内容 |
|------|----------|----------|
| 顶部导航栏 | 固定48-64px | Logo、系统名称、全局工具、用户入口 |
| 侧边栏 | 固定宽度200-260px | 导航菜单、模块分组 |
| 页面标题区 | 动态40-60px | 标题、面包屑、操作按钮组 |
| 内容区域 | 弹性填充 | 核心内容主体 |
| 底部栏 | 固定32-40px | 版权、版本信息 |

## 三、页面元素清单

### 3.1 导航元素

| 元素类型 | 使用场景 | 样式规范 |
|----------|----------|----------|
| 面包屑 | 显示当前位置层级 | `首页 / 模块 / 页面`，可点击跳转 |
| 标签页 | 同一页面内内容切换 | 最多5个标签，超出需滚动或下拉 |
| 步骤条 | 多步骤流程指引 | 当前步骤高亮，已完成可回跳 |

### 3.2 功能按钮

| 按钮类型 | 使用场景 | 样式规范 |
|----------|----------|----------|
| 主要按钮 | 页面核心操作（保存、提交） | 主色调填充，高优先级 |
| 次要按钮 | 辅助操作（取消、重置） | 边框样式，低优先级 |
| 文字按钮 | 低优先级操作（查看详情） | 无边框，仅文字 |
| 图标按钮 | 空间紧凑的操作（编辑、删除） | 仅图标，悬停显示提示 |
| 批量按钮 | 列表批量操作 | 需选中数据后启用，禁态灰色 |

### 3.3 数据展示组件

| 组件类型 | 使用场景 | 配置项 |
|----------|----------|--------|
| 数据表格 | 结构化数据列表 | 列配置、排序、分页、行操作 |
| 卡片列表 | 非结构化数据展示 | 卡片样式、字段布局、操作入口 |
| 详情面板 | 单条数据完整信息 | 分组显示、字段标签 |
| 统计卡片 | 关键指标展示 | 数值、趋势、图标 |
| 图表 | 数据可视化分析 | 类型选择、数据源配置 |

### 3.4 表单控件

| 控件类型 | 使用场景 | 配置项 |
|----------|----------|--------|
| 输入框 | 短文本输入 | 最大长度、占位提示、验证规则 |
| 文本域 | 长文本输入 | 行数、字数限制 |
| 下拉选择 | 选项集合选择 | 单选/多选、搜索、分组 |
| 日期选择 | 日期时间输入 | 格式、范围限制 |
| 开关 | 二值切换 | 启用/禁用状态文案 |
| 文件上传 | 文件附件 | 类型限制、大小限制、数量限制 |

### 3.5 状态提示

| 状态类型 | 使用场景 | 展示方式 |
|----------|----------|----------|
| 加载中 | 数据请求等待 | 遮罩层+旋转图标，或骨架屏 |
| 空数据 | 无数据结果 | 空状态图标+引导文案+操作按钮 |
| 错误提示 | 操作失败 | Toast消息或错误面板 |
| 成功提示 | 操作成功 | Toast消息，自动消失 |

## 四、交互规则

### 4.1 列表页交互

| 触发条件 | 响应行为 |
|----------|----------|
| 点击搜索 | 即时搜索或防抖500ms后触发 |
| 点击筛选 | 展开筛选面板，实时更新结果 |
| 点击行 | 进入详情页或展开行详情 |
| 批量选中 | 显示批量操作栏，启用批量按钮 |
| 点击排序 | 列头点击切换升序/降序 |
| 翻页 | 切换页码，保持筛选条件 |
| 刷新 | 重新加载当前页数据 |

### 4.2 表单页交互

| 触发条件 | 响应行为 |
|----------|----------|
| 输入失焦 | 触发字段验证，显示错误提示 |
| 点击提交 | 全表验证，通过后提交，失败定位首错 |
| 点击取消 | 弹窗确认是否放弃编辑 |
| 必填为空 | 禁用提交按钮，显示必填标记 |
| 表单脏检 | 离开页面提示保存变更 |

### 4.3 弹窗交互

| 弹窗类型 | 使用场景 | 操作按钮 |
|----------|----------|----------|
| 确认弹窗 | 删除、放弃等不可逆操作 | 确认 + 取消 |
| 信息弹窗 | 展示详情信息 | 关闭 |
| 表单弹窗 | 快速编辑、新增 | 保存 + 取消 |

## 五、数据字段规范

### 5.1 字段属性定义

| 属性 | 说明 | 取值 |
|------|------|------|
| 字段名称 | 用户可见的显示名 | 中文标签 |
| 字段标识 | 数据库/API字段名 | snake_case |
| 数据类型 | 字段值类型 | 文本/数字/日期/枚举/关联 |
| 是否必填 | 提交时是否必须 | 是/否 |
| 是否可编辑 | 用户是否可修改 | 是/否 |
| 显示格式 | 前端展示格式 | 日期YYYY-MM-DD等 |
| 默认值 | 初始填充值 | 具体值或表达式 |

### 5.2 常见显示格式

| 数据类型 | 格式示例 |
|----------|----------|
| 日期 | YYYY-MM-DD、YYYY-MM-DD HH:mm |
| 金额 | ¥1,234.56、保留两位小数 |
| 百分比 | 12.5%、最多一位小数 |
| 编号 | 自动生成、前缀+序号 |
| 状态 | 状态标签+颜色 |
| 关联 | 显示关联对象名称，可点击跳转 |

## 六、样式规范

### 6.1 色彩方案

| 用途 | 色值说明 |
|------|----------|
| 主色调 | 系统品牌色，用于主要按钮、高亮 |
| 成功色 | 绿色系，成功状态 |
| 警告色 | 橙色系，警告提示 |
| 错误色 | 红色系，错误提示 |
| 禁用色 | 灰色系，禁用状态 |
| 文本色 | 正文、标题层级 |

### 6.2 字体规范

| 类型 | 字号 | 用途 |
|------|------|------|
| 大标题 | 20-24px | 页面主标题 |
| 小标题 | 16-18px | 区块标题、卡片标题 |
| 正文 | 14px | 内容文本、表格内容 |
| 辅助文 | 12-13px | 提示说明、标签 |
| 微型文 | 11-12px | 时间戳、次要信息 |

### 6.3 间距规范

| 类型 | 数值 | 用途 |
|------|------|------|
| 大间距 | 24-32px | 区块分隔 |
| 中间距 | 16-20px | 组件间距 |
| 小间距 | 8-12px | 元素间距 |
| 微间距 | 4-6px | 紧凑排列 |

## 七、响应式适配

### 断点定义

| 设备类型 | 断点范围 | 布局调整 |
|----------|----------|----------|
| 桌面 | ≥1200px | 完整布局，侧边栏固定 |
| 平板 | 768-1199px | 侧边栏可折叠，表格简化 |
| 手机 | <768px | 侧边栏隐藏，卡片替代表格 |

### 8.4 交互规则
- 核心交互流程、特殊交互说明

### 8.5 数据字段
- 字段列表表格（名称、标识、类型、必填、格式）

### 8.6 样式说明
- 特殊样式需求、响应式适配要点
