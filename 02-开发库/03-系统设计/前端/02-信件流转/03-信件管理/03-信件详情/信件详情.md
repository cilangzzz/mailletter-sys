# 信件详情 - UI设计

## 一、基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 信件详情 |
| 页面标识 | letter-detail |
| 所属模块 | M01-信件核心模块 |
| 页面类型 | 详情页 |
| 路由路径 | /mailbox/letter/:id |

## 二、页面布局

### 2.1 整体布局框架

采用**三栏并列布局**，各栏内容独立展示，支持响应式调整为单栏堆叠。

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  页面标题区                                                                    │
│  - 信件编号：XX202604001                                                       │
│  - 信件状态：[状态标签]                                                         │
│  - 预警状态：[预警标签]（如有）                                                  │
│  - 操作按钮组：编辑 | 指派 | 审核 | 回复 | 删除                                 │
├──────────────────┬──────────────────┬──────────────────────────────────────────┤
│                  │                  │                                          │
│   左侧栏         │    中间栏        │           右侧栏                         │
│   留言信息       │    留言属性      │           处理信息                       │
│                  │    办理单位      │           批示意见                       │
│   宽度：30%      │    办理单位      │           本站答复                       │
│                  │    宽度：30%     │           宽度：40%                      │
│                  │                  │                                          │
│  ┌─────────────┐│ ┌─────────────┐ │ ┌────────────────────────────────────────┐│
│  │ 网民信件信息 ││ │ 接收单位填写 │ │ │ 指派处理链路                            ││
│  │             ││ │             │ │ │                                        ││
│  └─────────────┘│ └─────────────┘ │ │  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐      ││
│                 │                 │ │  │接收 │→│主办 │→│督办 │→│协办 │      ││
│  ┌─────────────┐│ ┌─────────────┐ │ │  │单位 │ │单位 │ │单位 │ │单位 │      ││
│  │ 网民用户信息 ││ │ 办理单位    │ │ │  └─────┘ └─────┘ └─────┘ └─────┘      ││
│  │             ││ │             │ │ │                                        ││
│  └─────────────┘│ └─────────────┘ │ └────────────────────────────────────────┘│
│                 │                 │                                          │
│                 │                 │ ┌────────────────────────────────────────┐│
│                 │                 │ │ 批示意见                                ││
│                 │                 │ │                                        ││
│                 │                 │ └────────────────────────────────────────┘│
│                 │                 │                                          │
│                 │                 │ ┌────────────────────────────────────────┐│
│                 │                 │ │ 本站答复                                ││
│                 │                 │ │                                        ││
│                 │                 │ └────────────────────────────────────────┘│
│                 │                 │                                          │
└──────────────────┴──────────────────┴──────────────────────────────────────────┘
```

### 2.2 区域说明

| 区域 | 宽度比例 | 包含内容 |
|------|----------|----------|
| 页面标题区 | 固定高度60px | 信件编号、状态标签、预警提示、操作按钮组 |
| 左侧栏 | 30% | 网民信件信息面板 + 网民用户信息面板 |
| 中间栏 | 30% | 接收单位填写面板 + 办理单位面板 |
| 右侧栏 | 40% | 指派处理链路 + 批示意见面板 + 本站答复面板 |

### 2.3 响应式适配

| 设备类型 | 布局调整 |
|----------|----------|
| 桌面（≥1200px） | 三栏并列显示 |
| 平板（768-1199px） | 左右两栏，左侧合并左中栏，右侧单独显示 |
| 手机（<768px） | 单栏堆叠，按顺序展示：留言信息 → 留言属性 → 处理信息 |

| 按钮名称 | 按钮类型 | 触发条件 | 操作结果 |
|----------|----------|----------|----------|
| 编辑 | 次要按钮 | 信件状态为待处理、当前用户为接收单位 | 打开编辑弹窗 |

## 四、交互规则

### 4.1 页面加载交互

| 触发条件 | 响应行为 |
|----------|----------|
| 进入页面 | 加载信件详情数据，显示骨架屏或加载动画 |
| 数据加载完成 | 各面板依次渲染，按从左到右顺序 |
| 数据加载失败 | 显示错误提示，提供刷新按钮 |

### 4.2 面板展开/折叠

| 触发条件 | 响应行为 |
|----------|----------|
| 点击面板标题 | 展开/折叠面板内容区域 |
| 默认状态 | 所有面板默认展开 |
| 折叠状态 | 面板标题栏保留，内容区域隐藏 |

### 4.3 操作按钮交互

| 按钮操作 | 前置条件 | 弹窗类型 | 弹窗内容 |
|----------|----------|----------|----------|
| 编辑 | 信件状态为待处理/受理中，用户为接收单位 | 表单弹窗 | 可编辑字段表单 |
| 指派 | 信件状态为待处理，用户为接收单位 | 表单弹窗 | 选择办理单位、角色 |
| 审核 | 信件状态为待审核，用户为审核角色 | 表单弹窗 | 审核意见、通过/不通过 |
| 回复 | 信件状态为受理中/待回复，用户为办理单位 | 表单弹窗 | 回复内容、是否结案 |
| 删除 | 用户为管理员 | 确认弹窗 | 删除原因输入、确认取消 |
| 收藏 | 所有用户 | 无弹窗 | 直接切换收藏状态 |
| 打印 | 所有用户 | 无弹窗 | 打印预览页面 |

### 4.4 查看流转详情

| 触发条件 | 响应行为 |
|----------|----------|
| 点击指派链路节点 | 弹窗显示该节点的详细流转信息 |
| 点击办理单位卡片 | 弹窗显示该单位的完整办理记录 |

### 4.5 附件操作

| 触发条件 | 响应行为 |
|----------|----------|
| 点击附件文件名 | 预览附件（图片/PDF直接预览） |
| 点击下载按钮 | 下载附件文件 |

### 4.6 权限控制交互

| 场景 | 响应行为 |
|------|----------|
| 用户无操作权限 | 对应按钮隐藏或禁用（灰色） |
| 用户无敏感信息权限 | 敏感字段脱敏显示，点击提示"权限不足" |
| 用户非办理单位 | 办理单位面板仅显示基本信息 |
