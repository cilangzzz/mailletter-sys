# M01 信件核心模块 - 实体设计

## 文档信息

**模块编号：** M01
**模块名称：** 信件核心模块
**文档版本：** v1.0
**创建日期：** 2026-04-13
**技术栈：** Java 17 + Spring Boot + MyBatis Plus

---

## 1. 实体类设计概述

### 1.1 实体类分类

| 分类 | 说明 | 基类 | 包路径 |
|------|------|------|--------|
| DO实体类 | 数据库实体对象，与表结构对应 | TenantBaseDO | dal/dataobject/letter |
| ReqVO类 | 请求参数对象 | 无 | controller/admin/letter/vo |
| RespVO类 | 响应结果对象 | 无 | controller/admin/letter/vo |

### 1.2 基类说明

**TenantBaseDO 基类字段：**

| 字段名 | 类型 | 说明 |
|--------|------|------|
| tenantId | Long | 租户ID |
| createTime | LocalDateTime | 创建时间 |
| updateTime | LocalDateTime | 更新时间 |
| creator | String | 创建者 |
| updater | String | 更新者 |
| deleted | Boolean | 是否删除（逻辑删除） |

    RE_MODIFY(9, "重新修改"),
    RE_PROCESS(10, "重新办理");

    public static final int[] ARRAYS = Arrays.stream(values()).mapToInt(LetterStatusEnum::getStatus).toArray();

    /**
     * 状态值
     */
    private final Integer status;

    /**
     * 状态名
     */
    private final String name;

    @Override
    public int[] array() {
        return ARRAYS;
    }

    /**
     * 判断是否可以修改
     */
    public boolean canModify() {
        return this != COMPLETED;
    }

    /**
     * 判断是否可以获取AI模板
