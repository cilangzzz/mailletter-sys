## 1. DO实体类设计

### 1.1 基类说明

所有DO实体类继承 `TenantBaseDO`，已包含以下基础字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| tenantId | Long | 租户编号（多租户隔离） |
| createTime | LocalDateTime | 创建时间 |
| updateTime | LocalDateTime | 更新时间 |
| creator | String | 创建者 |
| updater | String | 更新者 |
| deleted | Boolean | 是否删除（软删除标记） |
