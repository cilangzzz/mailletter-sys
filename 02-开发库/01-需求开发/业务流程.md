## 1. 信件处理主流程

### 1.1 流程概述

```
网民写信 → 接收单位接收 → 指派办理单位 → 办理单位处理 → 审核办理结果 → 回复网民
```

### 1.2 流程节点说明

| 序号 | 节点名称 | 操作角色 | 说明 |
|------|----------|----------|------|
| 1 | 网民写信 | 网民 | 通过厅长信箱、局长信箱、信访件、12389平台等渠道提交信件 |
| 2 | 接收单位接收 | 接收单位 | 审核信件内容，确认是否受理 |
| 3 | 指派办理单位 | 接收单位 | 将信件指派给主办单位、督办单位、协办单位 |
| 4 | 办理单位处理 | 主办单位/协办单位 | 调查核实，撰写办理结果和回复内容 |
| 5 | 审核办理结果 | 接收单位/督办单位 | 审核回复内容质量，决定是否通过 |
| 6 | 回复网民 | 接收单位 | 将审核通过的回复发送给网民 |

## 2. 信件状态流转

### 2.1 状态定义

| 状态名称 | 说明 |
|----------|------|
| 待处理 | 信件刚进入系统，等待接收单位处理 |
| 受理中 | 信件已被接收，正在办理过程中 |
| 无法受理 | 信件不符合受理条件，被标记为无法受理 |
| 待审核 | 办理单位已提交回复，等待审核 |
| 审核不通过 | 回复内容未通过审核，需要重新修改或办理 |
| 待回复 | 审核通过，等待正式回复网民 |
| 重新修改 | 回复内容需要修改后再次提交审核 |
| 重新办理 | 需要重新调查处理后再次提交 |
| 已完成 | 信件已回复网民，流程结束 |

### 2.2 状态流转路径

```
待处理 → 受理中 → 待审核 → 待回复 → 已完成
           ↓                    ↓
       无法受理            审核不通过 → 重新修改 → 受理中
                                     ↓
                                 重新办理 → 受理中
```

## 3. 指派流程

### 3.1 指派类型说明

| 指派类型 | 说明 | 适用场景 |
|----------|------|----------|
| 逐级指派 | 从上级到下级逐层指派，中间单位自动受理 | 多层级管理结构 |
| 直接指派 | 上级直接指派给最终办理单位 | 紧急信件或明确责任 |
| 改派处室 | 更换接收单位或办理单位 | 原单位无法办理 |
| 取消指派 | 撤销指派，信件回退到待处理状态 | 信件撤回或重新分配 |

### 3.2 指派规则

- 同一信件同一部门只能被指派一次
- 每封信件必须有且仅有一个主办单位
- 督办单位和协办单位可选，数量不限
- 指派后信件状态自动变为受理中

## 4. 审核流程

### 4.1 审核层级

| 审核层级 | 审核角色 | 说明 |
|----------|----------|------|
| 一级审核 | 督办单位/接收单位 | 初审回复内容 |
| 终审 | 接收单位 | 最终审核，决定是否回复网民 |

### 4.2 审核操作说明

| 操作名称 | 说明 | 结果状态 |
|----------|------|----------|
| 提交回复 | 办理单位提交回复内容 | 待审核 |
| 一级审核通过 | 初审通过 | 待终审 |
| 终审通过 | 最终审核通过 | 待回复 |
| 直接终审通过 | 跳过初审直接通过 | 待回复 |
| 审核不通过 | 回复内容不合格 | 审核不通过 |

### 4.3 退回操作说明

| 操作名称 | 说明 | 结果状态 |
|----------|------|----------|
| 下级退回-无法处理 | 办理单位无法处理该信件 | 回退给上级单位 |
| 上级退回-重新修改 | 回复内容需要修改 | 重新修改 |
| 上级退回-重新办理 | 需要重新调查处理 | 重新办理 |

## 5. 角色职责流程

### 5.1 接收单位职责流程

```
接收信件 → 判断是否受理 → 指派办理单位 → 监督办理进度 → 审核回复 → 回复网民
              ↓
         无法受理 → 标记原因 → 结束
```

### 5.2 主办单位职责流程

```
接收指派 → 调查核实 → 撰写回复 → 提交审核 → 等待审核结果
                                        ↓
                               审核不通过 → 修改回复 → 再次提交
                                        ↓
                               需重新办理 → 重新调查 → 再次提交
```

### 5.3 督办单位职责流程

```
接收指派 → 监督主办单位 → 催办提醒 → 初审回复 → 提交终审
```

### 5.4 协办单位职责流程

```
接收指派 → 配合调查 → 提供专业意见 → 协助主办单位完成办理
```
